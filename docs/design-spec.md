# Tidal Ink Tattoo Studio Design Spec

## Design Summary
Tidal Ink Tattoo Studio will use a premium dark editorial aesthetic with warm gold accents. The design should feel refined, immersive, and modern, balancing visual impact with clarity and usability.

## Visual Direction
- Backgrounds: charcoal, near-black, deep slate
- Accents: muted metallic gold
- Text: off-white and soft grey
- Atmosphere: premium, artistic, calm, confident

## Colour Palette
- Ink Black: `#0A0A0B`
- Deep Charcoal: `#121317`
- Slate: `#1B1E24`
- Soft Ivory: `#F3EFE7`
- Mist Grey: `#B8B2A7`
- Aged Gold: `#C8A96B`
- Warm Bronze Hover: `#9F7A3F`

## Typography
- Headings: elegant serif feel using a high-contrast display or refined serif stack
- Body: clean sans-serif for readability
- Heading style: uppercase or tight editorial spacing where appropriate
- Large section titles with generous spacing

## Layout Principles
- Mobile-first responsive design
- Wide containers on desktop with breathing room
- Strong vertical rhythm
- Alternating dark surface panels for section separation
- Card layouts for artists and gallery categories

## Components
### Header
- Sticky or anchored top navigation
- Wordmark style brand treatment
- Links: Home, Artists, Gallery, Booking, Contact
- Gold-outlined CTA button for Booking

### Hero
- Full-width dark hero
- Large headline
- Supporting copy
- Primary CTA: Book a Consultation
- Secondary CTA: View Artists

### Section Blocks
- Intro/value section
- Artist preview cards
- Gallery highlight grid
- Booking process timeline/cards
- Contact info panel

### Artist Cards
- Artist name
- Specialty
- Short bio
- Experience indicator
- CTA to book

### Gallery Cards
- Category title
- Descriptive caption
- Stylish framed image blocks or textured placeholders

### Booking Process
- Step 1: Share your idea
- Step 2: Consultation and artist match
- Step 3: Design approval and deposit
- Step 4: Session day
- Step 5: Aftercare

### Footer
- Studio name
- Location
- Navigation repeat
- Booking CTA
- Copyright line

## Page-by-Page Structure
### Home
- Hero
- Studio introduction
- Featured artists
- Portfolio preview
- Why choose Tidal Ink
- Booking CTA

### Artists
- Intro header
- Artist grid or stacked profiles
- Distinct specialties and personality
- Final booking CTA

### Gallery
- Intro copy
- Featured work categories: Fine Line, Blackwork, Ornamental, Custom Pieces
- Showcase grid
- CTA section

### Booking
- Consultation-first overview
- Process steps
- Policies: deposits, rescheduling, age requirement, preparation
- CTA and enquiry directions

### Contact
- Address and studio hours
- Email and Instagram placeholders
- Travel/location details
- Enquiry prompt

## Imagery Direction
- High-contrast black and grey tattoo imagery
- Detail shots, studio atmosphere, artist-at-work moments
- If no real imagery is available, use elegant gradient/image placeholder blocks with descriptive alt text avoided unless actual images are used

## Interaction Notes
- Subtle hover transitions
- Gold accent states on links and buttons
- No excessive animation
- Fast-loading static pages

## Accessibility
- WCAG AA contrast compliance
- One H1 per page
- Semantic landmarks
- Focus-visible styles for interactive elements

## Technical Notes
- Next.js static export
- Tailwind CSS styling
- Unoptimized images enabled for static export compatibility
- Metadata per page
- Robots and sitemap included
- Local business schema on key pages
